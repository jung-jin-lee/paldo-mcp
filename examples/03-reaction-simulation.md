# Example 3: 텍스트에 대한 N명 반응 시뮬레이션

## 시나리오

"내가 쓴 마케팅 카피에 대해 다양한 한국인 10명의 반응을 보고 싶어."

이건 `paldo-mcp`의 직접 기능은 아니지만, **Claude(또는 다른 LLM)와 조합하는 정석 패턴**입니다.

## 워크플로

1. `persona_panel`로 다양성 보장된 10명 샘플링
2. Claude가 각 페르소나의 관점에서 카피에 반응
3. 종합 인사이트

## Claude Code 대화

```
You: 아래 마케팅 카피에 대해 다양한 한국인 10명의 반응을 받아줘.

"매일 아침, AI가 당신의 글을 미리 읽어드립니다 — Personae"

Claude: 먼저 다양성 보장 패널을 구성하겠습니다.
        ↳ persona_panel({
            n: 10,
            stratify_by: "age_decade"
          })

[10명 페르소나 반환 — 20대부터 60대까지 분산]

Claude: 이제 각 페르소나의 관점에서 카피를 평가하겠습니다.

        [페르소나 1 — 27세 / 서울 / 광고기획자]
        반응: "Personae라는 이름이 직관적이지 않아 두 번 읽어야 했어요..."

        [페르소나 2 — 58세 / 부산 / 자영업]
        반응: "AI가 글을 읽는다는 게 무슨 뜻인지 잘 모르겠네요..."

        ...

        종합 인사이트:
        - 30대 이하: 'Personae' 네이밍의 시적 매력 인정, 다만 직관성 부족
        - 40대 이상: 'AI가 읽는다'의 가치를 즉시 파악하지 못함
        - 공통: '미리'라는 키워드가 가장 강력하게 작동
```

## 향후: 전용 도구 (v0.3+)

`@paldo/reactions` 같은 별도 패키지로, 위 워크플로를 단일 호출로 묶을 예정. 단 외부 LLM 호출 비용/정책이 얽혀서 별도 옵트인 패키지로 분리됩니다.

## 재현 가능한 실험

`seed`를 고정하면 같은 패널을 다시 구성할 수 있어, 카피 A/B를 같은 청중에게 테스트하는 실험이 가능합니다:

```json
{ "n": 10, "stratify_by": "age_decade", "seed": 42 }
```
